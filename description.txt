This mod aims to create the inverse or opposite of many themed items and enemies in the game.
So far it adds:
/ 105 Items
/ 25 Tiles
/ NPCs:
// 1 Boss(es)
// 1 Town NPC(s)
// 1 Enemie(s)
// 14 Critters
/ Mod Integration:
// Boss Checklist
// Census NPC Checklist
// Subworld Library
// Structure Helper
/ Extras:
// Menu Themes
// Alternate Soundtrack (Partial)
